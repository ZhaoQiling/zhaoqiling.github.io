CatalanNumber

发表于 2017-03-12 | 分类于 algorithm

原理

令 h(0) = 1, h(1) = 1, catalan数满足
h(n)= h(0)×h(n-1)+h(1)×h(n-2) + … + h(n-1)×h(0) (n>=2)

通项公式

h(n) = C(2n, n) / (n + 1) = (2n)! / ((n + 1)! × n!)
