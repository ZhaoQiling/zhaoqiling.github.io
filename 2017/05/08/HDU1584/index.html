HDU1584

发表于 2017-05-08 | 分类于 algorithm

题目大意

给定值为1到10的纸牌的顺序，每张牌只能放在大一的纸牌上，每次移动的代价为两张牌的差值的绝对值， 求最小代价。

题目思路

在不改变数组排列顺序的前提下生成全排列的题，当然还要带上剪枝和关于回溯的判断， 真的是考察dfs的好题。
